# TukarIDR
Dibuat dengan Android Studio

Anda suka binggung sekarang kurs Dollar ke IDR berapa? atau sebaliknya? Boleh Coba nih ini Aplikasi.

TukarIDR merupakan aplikasi penghitung KURS Rupiah secara Online.
